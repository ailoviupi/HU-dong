# HU-dong Chat

一个仿微信的网页聊天应用，支持发送文字、表情和图片。

## 功能特性

- 💬 实时消息发送和接收
- 😊 丰富的表情包支持
- 🖼️ 图片发送功能
- 🔍 聊天记录搜索
- 👥 多个聊天会话切换
- 📱 响应式设计，支持移动端
- 🌙 优雅的UI设计，仿微信风格

## 使用方法

1. 在浏览器中打开 `index.html` 文件
2. 或者访问部署的GitHub Pages页面
3. 点击左侧选择聊天对象
4. 在输入框输入消息或点击表情按钮选择表情
5. 点击图片按钮发送图片

## 技术栈

- HTML5
- CSS3
- JavaScript (原生)
- 无需后端，本地运行
